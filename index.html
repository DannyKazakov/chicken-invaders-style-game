About


Yo we are Daniel Kazakov and Itay Paikin.

We used JQuery in almost every process whether when we switch pages or start the game, getting input from the user and keeping the flow of the game.

All these helped us to get the values from the DOM elements in a very easy way.

Obiviously, we had several difficulties at the beginning of the development. We weren't familiar with javascript and html before this course so most of the time was spent on understanding how to work with JQuery syntax and understanding how the DOM and how to do visual pleasing elements when we are both (sadly) aren't so creative. Most of the time spent on trying to align the elements together so it would look appeasing to the eye, working with css and creating the spaceship and the chickens so they would look properly on screen.

The project was incredibly challenging but we've learned a lot and we'll keep developing our skills in web development.
